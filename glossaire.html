Glossaire Cybersécurité

🔐 XSS (Cross Site Scripting)

Attaque qui permet d’injecter du code JavaScript dans une page web vue par d'autres utilisateurs.

🕵️ MITM (Man In The Middle)

Un attaquant intercepte les communications entre deux parties sans qu’elles ne s’en rendent compte.
